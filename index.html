Caesar Cipher (13) Converter Converter

The Caesar Cipher is one of the simplest and oldest ciphers known. It consists in shifting each letter a certain number of places down the alphabet. For this exercise, we shift each letter 13 places of the alphabet. So, it would work this way:

  • A --> N
  • B --> O
  • C --> P

...and so on.

NOTE: This cipher only works with upper case letters.
